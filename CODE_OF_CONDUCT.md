# Contributor Covenant Code of Conduct

## 我们的承诺

在营造开放和欢迎的环境中，我们承诺使参与我们社区的所有人都能获得无骚扰体验，无论其年龄、体型、残疾、种族、性别特征、性别认同与表现、经验水平、教育背景或社会经济地位。

## 我们的标准

有助于创造积极环境的行为示例包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 注重对社区最有利的事情
- 表现出对其他社区成员的同情

成员不可接受的行为示例包括：

- 使用带有性暗示的语言或意象
- 骚扰、侮辱性评论、个人或政治攻击
- 公开或私下骚扰
- 发布其他人的私人信息，如地址或电子邮件，无明确许可
- 其他可被合理认为在专业环境中不适当的行为

## 执行

管理员负责澄清和执行我们的行为标准。他们将采取适当和公平的纠正措施来应对他们认为不适当、威胁、冒犯或有害的任何行为。

## 范围

本行为准则适用于所有社区空间，也适用于当个人代表社区在公共场所。

## 举报

如果你目睹或经历不可接受的行为，请通过发送电子邮件至 [contact@example.com] 与项目维护者联系。所有投诉将被保密处理。

## 后续行动

任何接到不符合本行为准则的报告的项目维护者将采取他们认为适当的行动。

## 参考

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.0。
