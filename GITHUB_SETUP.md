# github 仓库配置

## 关于此仓库 (About)

**项目名称**: IPSecVPN Traffic Classification (IPSec VPN 加密流量分类系统)

**描述**: 基于深度学习的IPSec VPN加密流量分类系统，支持FTP、IM、Web等应用协议识别。

选择: **MIT License**

## 仓库可见性

- ✅ Public (公开)
- ✅ Include all branches when exporting

## 分支保护规则 (Branch Protection)

为 `main` 分支启用：

1. **Settings** → **Branches** → **Add rule**
2. **Branch name pattern**: `main`
3. 启用以下选项：
   - ✅ Require a pull request before merging
   - ✅ Require approvals (至少1个)
   - ✅ Require status checks to pass before merging
   - ✅ Require branches to be up to date before merging
   - ✅ Include administrators

## 默认分支

- 设为: `main`

## Pull Request 的管理

- ✅ Allow squash merging
- ✅ Allow rebase merging  
- ✅ Allow auto-merge
- ✅ Automatically delete head branches

## 讨论区设置

建议启用Discussion用于社区交流：
- **Settings** → **Features** → **Discussions**

## GitHub Pages (可选)

如需部署文档网站：
- **Settings** → **Pages**
- Source: Deploy from a branch
- Branch: main / docs
